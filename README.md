# block-boby
draws ascii character faces, for example: 
```
 |||||||||||||||||

      ~     ~
      o     O
         -
        ___
```

or 
```

          ###################
          #                 #
          #     ~     ~     #
          #     0     o     #
          #        -        #
          #        _        #
          #                 #
          ###################

```
